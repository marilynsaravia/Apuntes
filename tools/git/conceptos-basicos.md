# Conceptos básicos de Git

## ¿Qué es Git?
Git es un sistema de control de versiones que permite guardar cambios en tu código y colaborar con otros desarrolladores.

## ¿Qué es GitHub?
Una plataforma donde puedes alojar tus repositorios Git en la nube.

## Conceptos importantes
- **Repositorio**: proyecto gestionado por Git.
- **Commit**: un cambio guardado en el historial.
- **Rama (branch)**: una línea de desarrollo separada.
- **Merge**: unir cambios de una rama a otra.
- **Remote**: repositorio que está en la nube (GitHub).
- **Origin**: nombre por defecto del remote principal.
