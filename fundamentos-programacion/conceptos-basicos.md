# Conceptos básicos de programación

Este documento reúne los principios fundamentales que comparten todos los lenguajes de programación.

## ¿Qué es un programa?
Un conjunto de instrucciones que el ordenador ejecuta para resolver un problema.

## Conceptos esenciales
- **Variables**: guardan información.
- **Tipos de datos**: números, texto, booleanos…
- **Operadores**: permiten comparar o realizar operaciones matemáticas.
- **Condicionales**: permiten tomar decisiones.
- **Bucles**: permiten repetir acciones.
- **Funciones**: agrupan instrucciones reutilizables.
- **Arrays**: almacenan listas de valores.

Estos conceptos son la base de cualquier lenguaje como JavaScript, PHP, C#, Python, etc.
